## 🔧 Common Issues and Fixes

### ❌ "Auto farm stops after the first wave"
**Fix:** This usually occurs when the wave completion trigger changes after a game update. Toggle **Auto Farm** off and back on to reset the wave state detector. If the issue persists across multiple toggles, download the latest version from the [Releases page](https://demnarena.github.io/demon-arena/) — wave completion handlers are updated with every Demon Arena game patch.

---

### ❌ "GUI doesn't appear after executing"
**Fix:** Ensure *Demon Arena* is fully loaded — your character must be active in the arena with the HUD visible before you execute. Press **Right Shift** (default GUI toggle key) if the panel doesn't appear automatically. If that fails, rejoin the server and execute after a clean fresh load.

---

### ❌ "Insta kill isn't working on boss enemies"
**Fix:** Some boss enemies in *Demon Arena* use a protected health system that requires the insta kill handler to target a specific internal value. If a boss isn't dying in one hit, go to *Settings* → *Insta Kill* → enable **Boss Override Mode**. This activates the extended boss health bypass that covers all protected enemy types.

---

### ❌ "God mode stops working after taking a specific damage type"
**Fix:** Demon Arena periodically introduces new damage types (void damage, curse damage) that require separate god mode handlers. If a new damage type is bypassing god mode, download the latest script version — damage type coverage is updated within 24–48 hours of each game patch.

---

### ❌ "Auto rebirth is triggering too early / at the wrong time"
**Fix:** Open *Settings* → *Auto Rebirth* → *Trigger Threshold* and adjust the minimum stat value or wave count required before rebirth is allowed. The default threshold targets optimal stat compounding ratios, but this can be customized based on your preferred progression pace.

---

### ❌ "Auto collect currency is missing some drops"
**Fix:** Very fast wave clears (especially with insta kill active) can generate drops faster than the default collection scan rate. Go to *Settings* → *Auto Collect* → *Scan Rate* and increase it to *Fast* or *Instant* mode. This increases the frequency of drop detection sweeps to match the accelerated clear speed.

---

### ❌ "Script errors on mobile during execution"
**Fix:** Verify your mobile executor is fully compatible with this **demon arena script**. Delta iOS and Ronix are the tested and recommended options. If using an alternate executor, check its UNC compatibility rating. Also verify the full script was copied without truncation — partial script pastes cause silent parse errors on mobile execution environments.
